# GoftarNAP

**Version 1.0.0**

A simple and clean document viewer developed by **Aripa Studio**.

---

## About

**GoftarNAP** is a lightweight application designed to open and read structured JSON documents in a clean, document-style interface.

The main goal of this software is **simplicity**.

---

## How to Use

1. Open the application.
2. Click on **Add New Document**.
3. Select your JSON file.
4. The file will appear in the document list.
5. Click on the file name and then click **Open Document** to start reading.

---

## Future Plans

In upcoming versions, we will add a built-in **Document Builder** — a simple tool that helps you easily create JSON files compatible with GoftarNAP.

We will also continue improving the user interface while keeping the software simple and easy to use.
